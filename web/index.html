Deployment notes

What this skeleton does and does not do

  • GitHub Pages hosts this static dashboard only.
  • The API still needs a separate .NET-capable host and a writable SQLite volume.
  • The current API enables cross-origin GET requests so this dashboard can read from it.
  • Write/import endpoints need a tighter CORS and credential story before they ship.
